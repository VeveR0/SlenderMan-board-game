# About

This game was developed during my second year of learning programming.

# Rules

*Collect all the notes, survive, and escape!*

The player must collect all 5 notes hidden around the camp before escaping.

Player wins: if they collect all the notes and escape.

Player loses: if they escape without collecting all the notes or are killed by Slender Man.

# Controls and Usage

<img width="848" height="428" alt="image" src="https://github.com/user-attachments/assets/eb1959a0-0bcf-4092-bf65-9c29e67b0992" />

## Controls

- `W` – Move up
- `A` – Move left
- `S` – Move down
- `D` – Move right

## Map Legend

- `1` = Player
- `I`, `_` = Map boundaries / fence
- `Y` = Trees (randomly generated)
- `#` = Cabin walls
- `` ` `` = Cabin doors
- `*` = Slender Man
- `/`, `\`, `|`, `-` = Slender Man's killing radius
- `?` = Notes
- `!` = Notes located inside Slender Man's killing radius

## Game States

- The map turns `green` when the player wins (the player collects all the notes and escapes)
- The map turns `red` when the player loses (the player gets killed by Slender Man or escapes, but doesn't collect all the notes)
